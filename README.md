🔐 SSH Log Monitoring & Brute Force Detection

------------------------------------------------------------------------

🛡️ PROJECT OVERVIEW

This project demonstrates how to monitor SSH login activity and detect
suspicious or brute-force login attempts in Kali Linux using Systemd
Journal logs.

It simulates failed SSH login attempts and analyzes system logs using
Linux commands and a custom Bash monitoring script.

------------------------------------------------------------------------

🎯 OBJECTIVES

-   Install and configure SSH server
-   Generate failed login attempts
-   Monitor authentication logs
-   Detect suspicious SSH activity
-   Extract and count attacker IP addresses

------------------------------------------------------------------------

🛠️ TOOLS & TECHNOLOGIES

-   Kali Linux
-   OpenSSH Server
-   systemctl
-   journalctl
-   Bash scripting
-   grep, awk, sort, uniq

------------------------------------------------------------------------

🚨 SECURITY CONCEPTS DEMONSTRATED

-   SSH Brute Force Detection
-   Username Enumeration Detection
-   Log Monitoring & Analysis
-   Basic Threat Detection Logic
-   SOC-Level Monitoring Fundamentals

-   Bash scripting for security automation
-   Blue Team fundamentals
